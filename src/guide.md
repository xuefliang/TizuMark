## 快速上手

### 界面一览

TizuMark 的界面分为四个主要区域：

| 区域 | 位置 | 说明 |
|------|------|------|
| **顶部工具栏** | 顶部 | 文件操作、格式插入、视图切换、主题、帮助 |
| **侧边栏** | 左侧 | 大纲导航与文件树（打开文件夹后） |
| **编辑区** | 中部 | CodeMirror 编辑器，语法高亮、自动补全 |
| **预览区** | 右侧 | 实时渲染 Markdown，同步滚动 |

### 基本操作

| 操作 | 方法 | 快捷键 |
|------|------|--------|
| 新建文件 | `文件 → 新建` | <kbd>Ctrl</kbd> + <kbd>N</kbd> |
| 打开文件 | `文件 → 打开`（支持批量选择） | <kbd>Ctrl</kbd> + <kbd>O</kbd> |
| 打开文件夹 | `文件 → 打开文件夹` | — |
| 保存文件 | `文件 → 保存` | <kbd>Ctrl</kbd> + <kbd>S</kbd> |
| 另存为 | `文件 → 另存为` | — |
| 关闭标签 | 点击标签 × 或右键菜单 | <kbd>Ctrl</kbd> + <kbd>W</kbd> |

> 直接将 `.md` 文件**拖入窗口**即可打开，支持多文件拖放。

## 编辑器功能

### 视图模式

工具栏中间的两个标签可在两种视图间切换：

- **预览模式** — 全屏预览文档，适合阅读和演示
- **编辑模式** — 左写右览，所见即所得

在编辑模式下：

| 操作 | 方法 |
|------|------|
| 折叠编辑器 | 点击左侧 <kbd>◄</kbd> 按钮 |
| 折叠预览 | 点击右侧 <kbd>►</kbd> 按钮 |
| 调整比例 | 拖动中间分隔条 |

### 侧边栏：大纲与文件

点击 `视图 → 侧边栏` 可显示或隐藏侧边栏。侧边栏包含两个标签页：

- **大纲**：自动显示文档标题结构（H1 ~ H6），按层级缩进。**点击任意标题**，预览区自动跳转并居中；大纲随编辑实时更新。
- **文件**：当使用 `文件 → 打开文件夹` 打开一个目录后，这里会以树形结构展示该目录下的文件。点击文件即可在标签页打开；目录树会监听文件夹的外部增删变化并自动刷新。

### 多标签页

| 操作 | 方法 |
|------|------|
| 新建标签 | 点击标签栏 <kbd>+</kbd> 或 <kbd>Ctrl</kbd> + <kbd>N</kbd> |
| 切换标签 | <kbd>Ctrl</kbd> + <kbd>Tab</kbd> / <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Tab</kbd> |
| 关闭标签 | 点击 × 或 <kbd>Ctrl</kbd> + <kbd>W</kbd> |
| 右键菜单 | 关闭 / 关闭其他 / 关闭所有 / 复制路径 |
| 双击新建 | 标签栏空白处双击 |

已修改未保存的标签会显示 `*` 标记。

### 查找与替换

两套独立搜索系统，各司其职：

**编辑器查找**（<kbd>Ctrl</kbd> + <kbd>F</kbd> / <kbd>Ctrl</kbd> + <kbd>H</kbd>）：

| 功能 | 说明 |
|------|------|
| 普通查找 | 输入关键词，点击下一个 / 上一个，并显示匹配数量 |
| 替换 | 输入新内容，点击替换或全部替换 |
| 区分大小写 | 精确匹配大小写 |
| 正则表达式 | 支持 JavaScript 正则语法 |

**预览查找**（预览模式下 <kbd>Ctrl</kbd> + <kbd>F</kbd>）：在渲染后的内容中搜索并高亮定位。

### 右键菜单

三处右键菜单，减少鼠标往返：

- **编辑区右键**：剪切 / 复制 / 粘贴 / 结构插入 / 文本格式 / 列表 / 链接与媒体 / 查找替换 / 全选
- **预览区右键**：复制 / 全选 / 复制为 HTML / 在预览中查找
- **标签页右键**：关闭 / 关闭其他 / 关闭所有 / 复制文件路径

### 外部文件变更提示

当打开的文件在 TizuMark 之外被其他程序修改时，顶部会弹出提示横幅，提供「重新加载」「忽略」「全部重新加载」「全部忽略」选项，避免误覆盖你的修改。

### 大文档保护

文档过大（超过约 5000 行或 4MB）时，预览会采用滑动窗口只渲染当前阅读区域，并在编辑区顶部提示，保证大文件依然流畅不卡顿。

## 插入功能详解

### 结构插入

| 元素 | 插入内容 |
|------|----------|
| 标题 H1 ~ H6 | `#` ~ `######` 前缀标题 |
| 代码块 | ` ``` ` 包围的代码块（支持 JavaScript / Python / Rust / HTML / CSS / YAML / Shell 等语法高亮） |
| 表格 | 3×3 基础表格模板 |
| 引用块 | `>` 前缀引用段落 |
| 提示块 | Note / Tip / Warning / Caution / Important |
| 数学公式 | `$$` 包围的独立公式（KaTeX 渲染） |
| Mermaid 图表 | 流程图 / 时序图等模板 |
| 水平线 | `---` 分割线 |
| 目录 | `[TOC]` 标记，自动生成文档目录 |

> 行内公式使用 `$...$`，独立公式使用 `$$...$$`。含 `&` 的公式会被自动转义处理。

### 文本格式

| 格式 | 语法 | 效果 |
|------|------|------|
| 加粗 | `**文本**` | **粗体文本** |
| 斜体 | `*文本*` | *斜体文本* |
| 删除线 | `~~文本~~` | ~~删除文本~~ |
| 行内代码 | `` `code` `` | `代码片段` |
| 高亮 | `==文本==` | ==高亮文本== |
| 上标 | `<sup>文本</sup>` | ^上标^ |
| 下标 | `<sub>文本</sub>` | ~下标~ |

### 列表

- 无序列表 —— `- ` 前缀
- 有序列表 —— `1. ` 前缀
- 任务列表 —— `- [ ] ` 前缀（预览中可勾选）

### 链接与媒体

- 链接 —— `[文本](URL)`，快捷键 <kbd>Ctrl</kbd> + <kbd>K</kbd>
- 图片 —— `![描述](图片URL)`

### 提示块（Callout）

TizuMark 支持 GitHub 风格的提示块，用于在文档中突出重要信息：

```markdown
> [!NOTE]
> 这是一个普通提示信息。

> [!TIP]
> 这是一个实用建议或技巧。

> [!WARNING]
> 这是一个需要关注的警告信息。

> [!CAUTION]
> 这是一个需要特别注意的风险提示。

> [!IMPORTANT]
> 这是一个至关重要的信息。
```

## 图片管理

TizuMark 提供完善的图片处理能力，支持多种插入方式和自动去重。

### 插入图片

| 方式 | 操作 | 说明 |
|------|------|------|
| 粘贴 | <kbd>Ctrl</kbd> + <kbd>V</kbd> | 粘贴剪贴板中的图片（截图、复制图片文件等） |
| 插入对话框 | `插入 → 图片` | 支持选择本地文件或输入网络图片 URL |
| 拖拽 | 拖入图片文件 | 将图片文件从文件管理器拖入编辑区 |

### 自动去重

同一内容的图片只会存储一份。TizuMark 通过 **MD5 哈希**比对图片内容：

- 粘贴或插入图片时，自动计算文件 MD5
- 如果该图片已存在，直接复用已有文件
- 同名但内容不同的图片不会冲突

### 图片存储方式

可在 `文件 → 设置 → 图片存储方式` 中配置：

- **复制到 assets/（推荐）**：图片保存为独立文件，md 文件轻量，便于版本管理
- **Base64 嵌入**：图片编码到 md 文件内，单文件即可分享，但体积显著增大（约原图 1.4 倍）

### 图片存储路径

可在 `文件 → 设置 → 图片存储路径` 中配置：

- **相对路径**（默认）：相对于当前 Markdown 文件所在目录
- **绝对路径**：使用设定的固定目录（如 `D:\assets`）

设置下方会动态显示实际的引用路径示例，方便确认。

### 自动宽高属性

插入图片时自动附加原始宽高信息：

```html
<img src="assets/abc123.png" width="800" height="600" alt="示例图片">
```

你可以直接在编辑器中修改或删除 `width`/`height`：

- 改为百分比：`width="100%"`
- 完全删除：编辑器会按原始尺寸渲染预览

## 自动更新

TizuMark 启动时会在后台静默检查新版本：

| 场景 | 行为 |
|------|------|
| 启动时检查 | 静默执行，无弹框、无提示 |
| 发现新版本 | 自动弹出更新对话框，显示版本号、发布说明、下载按钮 |
| 已是最新 | 什么都不做 |
| 手动检查 | `帮助 → 检查更新`，有更新则弹框，无更新则 Toast 提示 |
| 更新失败 | Toast 提示"检查更新失败" |

更新对话框：

- 显示当前版本和新版本
- 展示发布说明（Markdown 渲染）
- 点击「下载更新」→ 下载进度条 → 「立即安装」→ 安装并重启

## 导出功能

### 导出 HTML

`文件 → 导出 HTML` 生成独立的 HTML 网页文件：

- 包含完整 CSS 样式
- 表格、代码块、数学公式、Mermaid 图表均保留样式
- 可直接在浏览器中打开分享

### 导出长图

`文件 → 导出长图` 生成高清 PNG 长截图：

- 宽度固定 800px，高度自适应
- 支持暗黑主题样式
- 适合发布到社交媒体

### 导出 PDF

`文件 → 导出 PDF`（快捷键 <kbd>Ctrl</kbd> + <kbd>P</kbd>）通过系统打印对话框导出：

- 弹出提示确认后，使用浏览器原生打印功能
- 在打印对话框中可选择「另存为 PDF」、调整纸张方向/边距等
- 数学公式与 Mermaid 图表会被重新渲染以保证清晰度

| 设置项 | 可选值 | 默认值 | 说明 |

| 设置项 | 可选值 | 说明 |

- 点击「添加字体…」选择本地字体文件（如 `.ttf` / `.otf` / `.woff`），导入后可反复使用
- 导入的字体会出现在「编辑器字体」「预览字体」下拉框中，可分别指定
- 下方提供字体预览，方便对比效果

## 常见问题

### 如何恢复默认设置？

`文件 → 设置` 或 `文件 → 快捷键设置` 中点击「恢复默认」。

### 支持哪些文件格式？

`.md`、`.markdown`、`.txt`。更多格式支持计划中。

### 如何自定义快捷键？

`文件 → 快捷键设置`，点击「修改」后按下新组合键。点击「清除」可移除，点击「恢复默认」可还原。

### 预览区公式没有渲染？

检查语法：行内公式 `$...$`，独立公式 `$$...$$`。含 `&` 的公式会被自动转义处理。

### 导出长图 / PDF 没有包含图片？

确保图片可访问。本地文件使用相对路径，网络图片确保网络连接正常。

### 如何打开整个文件夹批量管理文件？

`文件 → 打开文件夹` 选择目录后，侧边栏「文件」标签会以树形展示目录内容，点击即可打开，并在外部改动后自动刷新。
