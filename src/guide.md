# TizuMark 使用说明

> 完整掌握 TizuMark 的各项功能，从入门到精通

---

## 快速上手

### 界面概览

TizuMark 的界面分为四个主要区域：

| 区域 | 位置 | 说明 |
|------|------|------|
| **工具栏** | 顶部 | 文件操作、插入、视图切换、主题切换、帮助 |
| **标签栏** | 工具栏下方 | 多标签页管理，支持同时编辑多个文档 |
| **编辑区** | 左侧（编辑模式） | CodeMirror 编辑器，语法高亮、自动补全 |
| **预览区** | 右侧（编辑模式） | 实时渲染 Markdown，同步滚动 |

### 基本操作

| 操作 | 方法 | 快捷键 |
|------|------|--------|
| 新建文件 | `文件 → 新建` | `Ctrl+N` |
| 打开文件 | `文件 → 打开`（支持批量选择） | `Ctrl+O` |
| 保存文件 | `文件 → 保存` | `Ctrl+S` |
| 另存为 | `文件 → 另存为` | — |
| 关闭标签 | 点击标签 × 或右键菜单 | `Ctrl+W` |

> **提示**：直接将 `.md` 文件拖入窗口也可打开，支持多文件拖放。

## 编辑器功能

### 视图模式

TizuMark 提供两种视图模式，通过工具栏中间的切换按钮控制：

- **预览模式** — 隐藏编辑器，全屏预览文档。适合阅读和演示
- **编辑模式** — 显示编辑器和预览双栏。左写右览，所见即所得

在编辑模式下：
- 点击编辑器左侧的 **◄** 按钮可折叠编辑器，全屏预览
- 点击预览区右侧的 **►** 按钮可折叠预览，全屏编辑
- 拖动中间分隔条可调整双栏比例

### 大纲导航

大纲功能帮助你在长文档中快速定位：

1. 点击 `视图 → 大纲` 开启左侧边栏
2. 自动识别文档中的 H1~H6 标题，按层级缩进显示
3. **点击任意标题**，预览区自动跳转并居中显示
4. 大纲随编辑内容实时更新

### 多标签页

TizuMark 支持多标签页编辑，提高工作效率：

| 操作 | 方法 |
|------|------|
| 新建标签 | 点击标签栏右侧 **+** 按钮，或 `Ctrl+N` |
| 切换标签 | 点击标签，或 `Ctrl+Tab` / `Ctrl+Shift+Tab` |
| 关闭标签 | 点击标签上的 ×，或 `Ctrl+W` |
| 右键菜单 | 在标签上右键：关闭 / 关闭其他 / 关闭所有 / 复制文件路径 |
| 批量关闭 | 右键菜单选择「关闭其他」或「关闭所有」 |
| 双击新建 | 在标签栏空白处双击新建标签 |

已修改但未保存的标签会在名称后显示 `*` 标记。

### 查找与替换

TizuMark 提供编辑器查找和预览查找两套独立的搜索系统：

**编辑器查找**（`Ctrl+F` / `Ctrl+H`）：

| 功能 | 说明 |
|------|------|
| 普通查找 | 输入关键词，点击「下一个」/「上一个」 |
| 替换 | 在替换输入框输入新内容，点击「替换」或「全部替换」 |
| 区分大小写 | 勾选后精确匹配大小写 |
| 正则表达式 | 支持 JavaScript 正则语法 |

**预览查找**（预览模式下 `Ctrl+F`）：直接在渲染后的内容中搜索并高亮定位。

### 右键菜单

TizuMark 在三个区域提供了便捷的右键菜单，减少鼠标往返操作。

#### 编辑器右键菜单

在编辑区右键，可快速访问以下功能：

| 菜单项 | 说明 |
|--------|------|
| **剪切** | 剪切选中文本 |
| **复制** | 复制选中文本 |
| **粘贴** | 粘贴剪贴板内容 |
| **结构插入** | 级联菜单：标题 / 代码块 / 表格 / 引用块 / 提示块 / 数学公式 / Mermaid 图表 / 水平线 / 目录 |
| **文本格式** | 级联菜单：加粗 / 斜体 / 删除线 / 行内代码 / 高亮 / 上标 / 下标 |
| **列表** | 级联菜单：无序列表 / 有序列表 / 任务列表 |
| **链接与媒体** | 级联菜单：链接 / 图片 |
| **查找替换** | 打开查找替换面板 |
| **全选** | 全选编辑器内容 |

> 右键菜单的结构插入、文本格式、列表、链接与媒体与顶部「插入」菜单完全一致，习惯用右键的用户无需移回工具栏。

#### 预览区右键菜单

在预览区右键，可执行以下操作：

| 菜单项 | 说明 |
|--------|------|
| **复制** | 复制选中的预览内容 |
| **全选** | 全选预览区内容 |
| **复制为 HTML** | 将选中内容以 HTML 格式复制到剪贴板，粘贴到 Word / 飞书 / 邮件等富文本编辑器时保留样式 |
| **在预览中查找** | 打开预览查找面板，在渲染后的内容中搜索关键词 |

#### 标签页右键菜单

在标签栏中的任意标签上右键：

| 菜单项 | 说明 |
|--------|------|
| **关闭** | 关闭当前标签页 |
| **关闭其他** | 保留当前标签，关闭所有其他标签 |
| **关闭所有** | 关闭全部标签页 |
| **复制文件路径** | 将标签对应文件的完整路径复制到剪贴板 |

## 插入菜单

工具栏「插入」按钮提供丰富的 Markdown 元素快速插入。所有插入操作也可通过编辑器右键菜单访问。

### 结构插入

| 元素 | 插入内容 |
|------|----------|
| **标题 H1~H6** | `#` ~ `######` 前缀标题 |
| **代码块** | ` ``` ` 包围的代码块（默认 JavaScript） |
| **表格** | 3×3 基础表格模板 |
| **引用块** | `>` 前缀引用段落 |
| **提示块** | Note / Tip / Warning / Caution / Important 提示框 |
| **数学公式** | `$$` 包围的独立公式 |
| **Mermaid 图表** | Mermaid 流程图模板 |
| **水平线** | `---` 分割线 |
| **目录** | `[TOC]` 标记（自动生成文档目录） |

### 文本格式

| 格式 | 语法示例 | 效果 |
|------|----------|------|
| 加粗 | `**文本**` | **粗体文本** |
| 斜体 | `*文本*` | *斜体文本* |
| 删除线 | `~~文本~~` | ~~删除文本~~ |
| 行内代码 | `` `code` `` | `代码片段` |
| 高亮 | `==文本==` | ==高亮文本== |
| 上标 | `<sup>文本</sup>` | ^上标^ |
| 下标 | `<sub>文本</sub>` | ~下标~ |

### 列表

- **无序列表** — `- ` 前缀
- **有序列表** — `1. ` 前缀
- **任务列表** — `- [ ] ` 前缀（预览中可点击勾选）

### 链接与媒体

- **链接** — `[文本](URL)`
- **图片** — `![描述](图片URL)`

## 提示块（Callout）

TizuMark 支持 GitHub 风格的提示块，用于在文档中突出重要信息：

```markdown
> [!NOTE]
> 这是一个普通提示信息。

> [!TIP]
> 这是一个实用建议或技巧。

> [!WARNING]
> 这是一个需要关注的警告信息。

> [!CAUTION]
> 这是一个需要特别注意的风险提示。

> [!IMPORTANT]
> 这是一个至关重要的信息。
```

## 导出功能

### 导出 HTML

`文件 → 导出 HTML` 生成独立的 HTML 网页文件：
- 包含完整 CSS 样式
- 表格、代码块、数学公式均保留样式
- 可直接在浏览器中打开分享

### 导出长图

`文件 → 导出长图` 生成高清 PNG 长截图：
- 宽度固定 800px，高度自适应内容
- 支持导出暗黑主题的样式
- 适合发布到社交媒体

| 设置项 | 可选值 | 默认值 | 说明 |
|--------|--------|--------|------|
| 字体大小 | 12~20px | 14px | 编辑器代码字体大小 |

### 预览设置

| 设置项 | 可选值 | 默认值 | 说明 |
|--------|--------|--------|------|
| 正文字号 | 14~20px | 16px | 预览区正文字体大小 |

> 点击工具栏右侧的太阳/月亮图标也可快速切换主题。

| 设置项 | 可选值 | 说明 |
|--------|--------|------|
| 默认视图 | 预览 / 编辑 | 启动时默认显示的视图模式 |

`文件 → 设置 → 界面语言` 可选择：

## 常见问题

### 如何恢复默认设置？

在 `文件 → 设置` 中点击「恢复默认」，或 `文件 → 快捷键设置` 中点击「恢复默认」。

### 支持哪些文件格式？

`.md`、`.markdown`、`.txt`。更多格式支持计划中。

### 如何自定义快捷键？

`文件 → 快捷键设置`，点击对应功能的「修改」按钮，然后按下新的组合键。点击「清除」可移除快捷键。点击「恢复默认」可还原。

### 预览区的公式没有渲染？

请检查公式语法是否正确。行内公式使用 `$...$`，独立公式使用 `$$...$$`。如果公式中包含 `&` 符号，Rust 后端会进行 HTML 转义，KaTeX 在渲染前会自动还原，无需特殊处理。

### 导出的长图没有包含图片？

图片需要能被访问。对于本地文件，请使用相对路径。对于网络图片，确保导出时网络连接正常。
